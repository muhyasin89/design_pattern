#There is 3 types of Design Pattern
1. Creation
   - Prototype
   - Builder
   - Singleton
   - Factory

2. Structural
   - Proxy
   - Decorator
   - Facade
   - Adapter
   - Flyweight

3.Behavioural
   - Chain of responsibility
   - Iterator
   - State
   - Strategy
   - Observer
   - Visitor
   - Template Method
   - Command
   - Momento
   - Mediator
   
   
What about anti-pattern? 
anti pattern is just fancy name, it same as structural no pattern at all.
